# Магазин вышивки крестом


## Запуск сервиса

 - Переименовываем файл .env.local в .env
 - Запускаем сервис и БД в Докере `docker-compose up -d --build`
 - После запуска контейнеров в браузере переходим по адресу http://localhost:8080/ для доступа в публичный интерфейс
 - После запуска контейнеров в браузере переходим по адресу http://localhost:8080/admin/stats для доступа в админку

## Остановка сервиса

 - Останавливаем сервис и БД в Докере `docker-compose down`
